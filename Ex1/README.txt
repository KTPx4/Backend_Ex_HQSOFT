----------------------------------------------------
	Project wrote by ASP.NET Core API + MSSQL
----------------------------------------------------

1. Edit connection string in appsettings.json
2. Build project
3. Open "Package Manager Console"
4. Run "update-database" to update migrations
6. Run SQL code in folder "SQL Code"

5. Run project
